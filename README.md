# AI Research Paper Assistant (RAG)

This project builds a Retrieval Augmented Generation (RAG) system
that answers questions from research papers using vector search.

## Features
- PDF document ingestion
- Text chunking
- Embedding generation
- Semantic search
- Question answering

## Technologies
- Python
- Sentence Transformers
- Vector search
- RAG pipeline

## How to Run

1. Install dependencies
pip install -r requirements.txt

2. Open the notebook
rag_chatbot.ipynb

3. Upload your research paper and ask questions# endee-rag-chatbot
RAG chatbot using Endee vector database
